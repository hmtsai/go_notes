
What is package, module, and repository.

See the reference.
https://github.com/golang/go/wiki/Modules#quick-start

- A repository contains one or more Go modules.
- Each module contains one or more Go packages.
- Each package consists of one or more Go source files in a single directory.

In another words.

- package : is a directory where one or more go source files resides.
- module : can have multiple packages.
- repository : one reposotory contains one module is easier maintain
